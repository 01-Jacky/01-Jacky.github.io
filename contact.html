Internship Watch

A filterable table with many of the latest Computer Science internship posting from indeed, allowing you to quickly search and filter through them.

Backend is an AWS Lambda function scheduled to crawl through hundreds of indeed posting each day, uploading software engineering internship postings into DynamoDB. Postings in DB have a 30 day time to live as stale job becomes irrelevent. Frontend uses data tables to quickly allow you to short and filter through postings.

https://internship-watch.herokuapp.com/
